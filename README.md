# Barcode Scanner with Webserver

## First time setup:

```bash
$ pip3 install virtualenv
$ virtualenv venv
$ source venv/bin/activate
$ pip install pybuilder
$ pyb install_dependencies
$ pyb
```

## Requirements:
- a runnig Redis server unter localhost




## Hints:
